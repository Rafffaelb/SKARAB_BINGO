About This Project

Project Overview

This is the SKARAB_BINGO project, which uses the SKARAB (Square Kilometre Array Reconfigurable Application Board) platform for radio astronomy applications. The system processes astronomical signals using FPGA-based digital backends.

Key Components

  • SKARAB Hardware: FPGA platform with high-speed ADCs
  • CASPER Toolflow: Framework for FPGA design
  • Python Scripts: Control and data acquisition tools
  • Digital Backends: Spectrometers and signal processors
